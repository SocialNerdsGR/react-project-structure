# React project structure

An opinionated React project structure using the following libraries:

- React router
- Formik
- Axios
- React Query

## Tools: 
- Vite
- Vitest
- TypeScript
- React Testing Library
- PNPM

## Run
- pnpm install
- pnpm run dev

## Lint
- pnpm run lint

## Build 
- pnpm run build

## Test
- pnpm test
